# Active Support

#### 作用
Active Support 是 Rails 里的工具类库，它扩充了许多 Ruby 的标准函数库，因此许多人写 Rails 的时候常常会有疑惑，这个方法到底是 Rails 里面的还是 Ruby 里面的。当然，Ruby 也常常会把许多 Rails 的函数库集成进来。
